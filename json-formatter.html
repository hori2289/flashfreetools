Format JSON for readability, minify for production, or validate for syntax errors.

How to use

Paste any JSON. Format adds indentation for readability. Minify compacts it for production. Validate checks for errors without changing the output. All processing uses the browser's built-in JSON.parse — nothing leaves your device.
